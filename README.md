# 🚗 Road Accident Analysis Dashboard (Excel)

## 📊 Overview
This project presents an interactive dashboard built using Microsoft Excel to analyze road accident data in the UK for the years 2021–2022. The dashboard highlights key trends, causes, and environmental factors affecting casualties.

## 🎯 Objective
To analyze accident data and identify patterns that can help improve road safety and support data-driven decision-making.

## 🚀 Key Features
- Year-wise comparison of accident data (2021 vs 2022)
- Analysis of casualties by vehicle type
- Road type and road condition insights
- Urban vs rural accident distribution
- Time-based analysis (day/night conditions)

## 📈 Key Insights
- A significant number of casualties fall under the **slight injury category**
- **Cars** are responsible for the majority of accidents
- Total casualties **decreased from 2021 to 2022**
- Most accidents occur on **single carriageway roads**
- **Dry road conditions** account for the highest number of accidents
- Higher accident rates are observed in **urban areas**
- Majority of accidents occur during **dark conditions**

## 🛠 Tools Used
- Microsoft Excel
- Data Cleaning & Processing
- Pivot Tables
- Data Visualization (Charts & Dashboard Design)
